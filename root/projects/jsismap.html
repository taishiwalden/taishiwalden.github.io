Interactive Scrollytelling Map

As part of my undergraduate curriculum, I completed the Donald C. Task Force Program at the University of Washington. My team wrote a research analysis capstone providing a detailed blueprint for the creation of a coalition to center Palestinian human rights within the conflict and empower the Palestinian voice. I wanted to make this project more accessible to a wider audience by making an interactive webpage that allows readers to visualize geographically where these events are occuring. I utilized Mapbox GL JS libraries to accomplish this.

Mapbox GL JS is a client-side JavaScript library for building web maps and web applications with Mapbox's modern mapping technology. You can use Mapbox GL JS to display Mapbox maps in a web browser or client, add user interactivity, and customize the map experience in your application.

Tools: Visual Studio Code, Mapbox GL JS, GitHub, HTML, CSS, JavaScript.
